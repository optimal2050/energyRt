# Build the mCommReg closure mapping (recipe "closure")

Build the mCommReg closure mapping (recipe "closure")

## Usage

``` r
recipe_closure(scen, names, fmp)
```

## Arguments

- scen:

  scenario object.

- names:

  character vector of closure mapping names (expects `mCommReg`).

- fmp:

  function mapping a parameter name to its on-disk path.

## Value

updated scenario object.

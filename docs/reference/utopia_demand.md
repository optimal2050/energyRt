# UTOPIA electricity load shape

A deterministic relative electricity-load shape by slice, for both
teaching calendars (replaces the vignette's former random load curve).
[`utopia_profiles()`](https://energyRt.org/reference/utopia_profiles.md)
/ the vignette scale it by a region's annual demand and the slice shares
to get energy per slice.

## Usage

``` r
utopia_demand
```

## Format

A data.frame with columns `calendar` (`utopia_m12h24`/`utopia_seasons`),
`slice` and `load` (relative, mean ~1).

## See also

[`utopia_profiles()`](https://energyRt.org/reference/utopia_profiles.md),
[utopia_weather](https://energyRt.org/reference/utopia_weather.md),
[utopia_stock](https://energyRt.org/reference/utopia_stock.md)

# Function to find duplicated values in interpolated scenario.

Function to find duplicated values in interpolated scenario.

## Usage

``` r
findDuplicates(x)
```

## Arguments

- x:

  scenario or data.frame with data to check.

## Value

data.frame with duplicated values.

## Examples

``` r
if (FALSE) { # \dontrun{
findDuplicates(scen_BASE)
} # }
```
